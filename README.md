# Institute of Digital Risk (IDR) - Brand & Website System

## Overview
This repository contains the complete frontend architecture and brand identity assets for the Institute of Digital Risk (IDR). Designed for a high-consequence technology ecosystem, the platform represents IDR as the premier destination for cybersecurity, AI governance, and digital risk professionals.

## Brand Identity Rationale
The IDR brand identity centers around the **Geometric 3D Cube**.
- **Data Infrastructure**: The cube represents the physical and digital servers that hold the world's most critical data.
- **Layered Security**: The gradients and facets of the cube symbolize depth, defense in depth, and robust structural integrity.
- **Resilience**: The stark contrast between the "Pure Black" background and the "Premium Orange (#FF6A00)" highlights the dynamic nature of risk mitigation—alert, actionable, and vital.
- **Institutional Weight**: The minimalism echoes the quiet competence of deeply technical environments (like OpenAI or Stripe), stripped of unnecessary ornamentation.

## Project Structure
- `/assets`: Contains the premium SVG logos and future image/icon assets.
- `/css`: Contains Vanilla CSS utilizing CSS Variables, Grid, and Flexbox to create a responsive, framework-free experience.
- `/js`: Contains lightweight Vanilla JavaScript for performant scroll animations and navigation UI.
- `/figma`: Contains the technical design specifications and component architecture.

## How to Run
Simply open `index.html` in any modern web browser. No compilation or local server is strictly required as no external frameworks are utilized.
